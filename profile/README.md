# 🏴 Komunites

**⚔️ Where mispronunciation meets deployment.**

It all started when someone tried to say "Kubernetes" in Thai and what came out was _คอมมูนิเตส_ (Komunites). Instead of correcting ourselves, we leaned into it — and accidentally founded a dev org. 💥

We're a small unit of engineers who build things for fun and because they're useful. Sometimes both at the same time. 🛠️
